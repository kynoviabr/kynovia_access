# Product

This section captures product direction for Kynovia Access.

## Purpose

Kynovia Access is a multi-condominium SaaS for gatehouse access control, administration, auditability, and resident-facing access workflows.

## Primary Users

- Condominium administrators
- Gatehouse operators
- Residents and authorized visitors
- Kynovia support and operations teams

## Initial Product Boundaries

The current repository scope is platform scaffolding only. Product flows will be specified before implementation in this section.

Future product documents should cover:

- Tenant and condominium onboarding
- Gatehouse access workflow
- Resident and visitor access lifecycle
- Audit and incident review
- Notification and integration expectations
